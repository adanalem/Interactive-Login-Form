# Interactive-Login-Form
An interactive, animated login form featuring a responsive Panda mascot that reacts to user input for a fun and engaging authentication experience.
Features
Live Interaction: Mascot expresses happiness while the user types in the username.

Privacy-Focused: Mascot covers its eyes when the password field is focused.

Responsive UI: Styled with Tailwind CSS for a modern, mobile-friendly look.

Lightweight: Built using vanilla JavaScript and HTML5—no heavy dependencies.

🛠 Tech Stack
HTML5

Tailwind CSS

JavaScript
